# Walmart Sales Forcasting(Kaggle)
Project Overview:
This project analyzes historical sales data from 45 stores and multiple departments of Walmart to identify sales trends, seasonal patterns and the impact of holidays on sales.The project also includes a Power BI Dashboard to visualize store performance and weekly sales trends.

DataSet:
The dataset used in this project was obtained from Kaggle.
It contains approximately 6435 rows of sales data with information such as:
Store ID
Department
Date
Weekly Sales
Holiday Indicator
Temperature
Fuel Price
CPI
Unemployment Rate
Year
Month
Weeks
This dataset represents weekly sales data across multiple stores and departments.

Project Steps:
Data Collection from Kaggle
Data Cleaning and preprocessing using Python
Exploratory Data Analysis to understand trends
Feature engineering (Month, Year, Week)
Sales trend analysis and forecasting
Dashboard development using Microsoft Power BI Desktop

The Power BI dashboard includes:
Total Sales KPI
Average Weekly Sales KPI
Store Performance Comparison
Weekly Sales Trend Analysis
Monthly Sales Trend
Holiday vs Non-Holiday Sales Impact

Key Insights:
Holiday weeks show higher sales compared to normal weeks
Certain stores consistently generate higher sales
Sales trends reveal seasonal demand patterns
